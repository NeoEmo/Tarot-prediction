1.0 Версия таро Оракула (Java 25+)

В данный момент может выполнять два действия:
1)Гадание по старшим арканам (Major)
2)Гадание по всем картам таро (All)

Взможна различная табуляция команд от человека (major/Major/MAJOR или all/All/ALL)
Добавленны ASCII рисунки к картам
Добавлен Makefile для запуска из под консоли без надобности иметь IDE (Достаточно иметь Java25+[желательно всё таки 25, на ней точно работает])


Будущие идеи для следующих версий:
1)Перевод на Английский

2)Добавление новых гаданий (На себя, на человека, на то как пройдёт неделя итд.)

UPD README#1

Теперь есть два вида оракула, старый со старыми технологиями(Много строк, строковых массивов, и очень не читаемо) и новый (c добавлением stream, переносом всех строк в файл, из которых по итогу идёт чтение, а затем уже выводится через System.out.println). Работа над проектом всё так же идёт потихоньку. Но теперь всё идёт в сторону не CLI, а JavaFX. Переделываться он тут не будет, а будет залит отдельным репозиторием
